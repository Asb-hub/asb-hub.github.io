## 👩‍💻 À propos

Portfolio personnel développé entièrement en **HTML, CSS et JavaScript vanilla**, sans framework.  
Déployé sur **GitHub Pages**.

---

## ✨ Fonctionnalités

- 🎨 Design moderne avec thème **bleu · violet · noir · rouge**
- ✍️ Effet de frappe animé (typewriter)
- 🌌 Particules interactives en arrière-plan
- 📊 Barres de compétences animées au scroll
- 👁️ Animations au défilement (scroll reveal)
- 🖱️ Curseur personnalisé
- 📱 100% Responsive (mobile, tablette, desktop)
- 📧 Formulaire de contact fonctionnel via **Formspree**
